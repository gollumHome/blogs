mac 搭建Python 虚拟环境

原因：不想高混淆本的的mac运行环境
步骤:

  1. sudo pip install virtualenv

  2. 建立一个工作目录

  3. 创建一个虚拟环境env1

  4. 安装virtualenvwrapper
    注：Virtaulenvwrapper是virtualenv的扩展包，可以更方便地新增，删除，复制，切换虚拟环境。
    命令

  5. 创建一个文件夹，存放所有的虚拟环境·

  6. 配置环境变量

6.1 查看python 安装位置

vi ~/.bash_profile 增加
export WORKON_HOME=’~/.virtualenvs’

export VIRTUALENVWRAPPER_PYTHON=’/Users/mac/anaconda/bin/python3’

source /Users/mac/anaconda/bin/virtualenvwrapper.sh

退出，执行 source ~/.bash_profile

mkvirtualenv env1
成功后，当前路径前面就会有 (env1)

$ lsvirtualenv -b
env1
env2
切换虚拟环境：
$ workon env1
查看环境里安装了哪些包：
$ lssitepackages
进入当前环境：
$ cdvirtualenv
进入当前环境的site-packages：
$ cdsitepackages
$ cdsitepackages pip
复制虚拟环境：
$ cpvirtualenv env1 env3
Copying env1 as env3…
退出虚拟环境：
$ deactivate
删除虚拟环境：
$ rmvirtualenv env2
Removing env2…

$ workon env1
安装numpy
pip install numpy

安装scipy
$ pip install scipy
