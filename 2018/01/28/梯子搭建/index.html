vultr 科学上网

一.购买合适的VPS

1.注册地址:官方地址：https://www.vultr.com/

2.vultr账户充值与配置

  • 1>支付方式：1.信用卡2.paypay3.支付宝->推荐支付宝即可
  • 2>在Billing界面选择充值,10美元(最少)

    搭建VPS

    1.步骤如下
  • 1>选择vps节点，（日本，美国等）
  • 2>操作系统centos或其他系统
  • 3>服务器配置,5美元即可(土豪随意)

2.部署deplpy ，如下图所示,等待服务器状态为running

3.测试服务器与本地是否连通，ping 即可，如果不通，说明被墙，重新删除服务器，换个节点地址，多试下。亲测日本不行，LA可以

4.ss账号生成（xsell,putty）

  • 1>远程连接上服务器
  • 2>wget –no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks.sh
  • 3>wget –no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks.sh
  • 4>./shadowsocks.sh 2>&1 | tee shadowsocks.log
  • 注：注：提示你输入你的SS SERVER的账号，和端口。不输入就是默认。跑完命令后会出来你的SS客户端的信息

5.windows下在客户端

  • 1>Shadowsocks-4.0.6.zip
    地址：https://github.com/shadowsocks/shadowsocks-windows/releases
  • 2> 配置客户端信息
  • 3> 在电脑右下角任务栏找到ss图标，右键点击，点击启用系统代理就可以了

linux-SSR多用户版配置详解

  1. 先停止shadowsocks
    /etc/init.d/shadowsocks stop
  2. 编辑配置文件
    vi /etc/shadowsocks.json 
    {  
"server":"0.0.0.0",
"server_ipv6": "[::]",
"local_address":"127.0.0.1",
"local_port":1080,
"port_password":{
 "8989":"password1",//着里输入想要的端口和密码
 "8990":"password2"，
 "8991":"password3"
},
"timeout":300,
"method":"aes-256-cfb",//加密方式可以修改也可以不修改本人用chacha20
"protocol": "origin",//协议也是可以修改了本人用auth_sha1
"protocol_param": "",
"obfs": "plain",//这里很重要,免流的请注意必须修改,本人用http_simple
"obfs_param": "",
"redirect": "",
"dns_ipv6": false,
"fast_open": false,
"workers": 1
}
  3. 退出并保存文件
  4. 修改防火墙，供多个人使用 (centos 7 的防火墙为firewall)
    1> 停止防火墙：systemctl stop firewalld
    2>编辑public.xml文件
    vi /etc/firewalld/zones/public.xml
  5. 保存重启动防火墙和ssr
    1> systemctl start firewalld
    2> /etc/init.d/shadowsocks start
